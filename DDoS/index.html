What is a DDoS Attack

A DDoS or Distributed Denial of Service Attack is an attack that involves flooding a server with so many requests that it becomes overwhelmed and its performance is ultimately slowed down.

DDoS attacks are normally carried out with a network of computers. Usually, one machine doesn’t provide enough network traffic to impact a server’s efficiency, however hackers have found ways to get around this. A common tactic is to create something referred to as a botnet. A botnet consists of a group of bots, these bots are often individual devices which have been infected with malware.

Why don’t DDoS attacks work?

Even though DDoS attacks seem like they would be efficient there are several reasons why DDoS attacks aren’t always that effective. The biggest issue with this attack is that most servers will have a firewall. If a firewall detects a lot of traffic from the same devices it would simply be able to block requests from that IP Address. However, with hackers becoming more advanced, tools have been developed which tricks the firewall into thinking that each request is unique.
